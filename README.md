# Animal Shelter Data Analysis System (AVL Tree in C)

This project is a  animal shelter management and analysis system written in C using an AVL Tree data structure.

## Features

- AVL Tree based indexing
- Self-balancing rotations
- Linked list for duplicate animal names
- File-based data loading
- Animal search by name
- Display full animal index
- Identify the most popular animal based on donation amount

## Data Structures Used

- AVL Tree
- Linked List
- Dynamic Memory Allocation

## Files

main.c → Main program and menu system  
avltree.c → AVL tree implementation  
avltree.h → AVL tree header definitions
